Package ancestry implements a data structure for fast ancestry enquiries in binary tree.

Definition

Consider nodes A and B of a rooted tree. B is called a proper descendant of A if there is a unique simple path from A to B and also A is not B.

Hint

You should read about properties of depth-first search (CLRS 22.3) before implementing this package.

▹ Example

▾ Example

Code:

// Create BST:
//     6
//    / \
//   2   8
//  / \
// 1   3
T := &tree.Tree{
    Value: 6,
    Left:  &tree.Tree{Value: 2, Left: &tree.Tree{Value: 1}, Right: &tree.Tree{Value: 3}},
    Right: &tree.Tree{Value: 8},
}

A := ancestry.New(T)

fmt.Println(A.IsDescendant(2, 2)) // false, not a _proper_ descentant
fmt.Println(A.IsDescendant(2, 3)) // true
fmt.Println(A.IsDescendant(6, 3)) // true
fmt.Println(A.IsDescendant(1, 8)) // false

Output:

false
true
true
false

func New ¶ 

func New(T *tree.Tree) *Ancestry

New creates an instance of Ancestry for the given binary tree with unique integer nodes. The running time is O(N) for N nodes.

func (*Ancestry) IsDescendant ¶ 

func (A *Ancestry) IsDescendant(a, b int) bool

IsDescendant determines if node b is a proper descendant of node a. The running time is guaranteed to be Θ(1). Panics if the given nodes are not in the tree.
