func LCS ¶ 

func LCS(s1, s2 string) string

LCS returns the longest common subsequence of the given strings.

▹ Example

▾ Example

Code:

fmt.Println(strings.LCS("vintner", "writers"))
fmt.Println(strings.LCS("ABCD", "ACBAD"))

Output:

iter
ABD

func Segmentation ¶ 

func Segmentation(s string, isWord func(w string) bool) ([]string, error)

Segmentation breaks the given string into words. A 'word' is a non-empty string for which the given isWord function returns true. An error is returned if the segmentation is not possible.
