Overview ▹

Overview ▾

Package tsort implements topological sorting of directed graph.

See https://en.wikipedia.org/wiki/Topological_sorting for more details.

Restrictions

The sorting must be implemented non-recursively.

▹ Example

▾ Example

Code:

package tsort_test

import (
    "fmt"
    "hsecode.com/stdlib/graph"
    "hsecode.com/stdlib/graph/tsort"
)

type Int int

func (v Int) ID() int {
    return int(v)
}

func Example() {
    G := graph.New(graph.Directed)

    for i := 0; i <= 3; i++ {
        G.AddNode(Int(i))
    }
    G.AddEdge(2, 1, nil)
    G.AddEdge(1, 0, nil)
    G.AddEdge(3, 1, nil)
    G.AddEdge(3, 0, nil)

    nodes, _ := tsort.New(G)
    fmt.Println(nodes)
    // Output: [3 2 1 0]
}

func New ¶ 

func New(g *graph.Graph) ([]graph.Node, error)

New returns one of the possible topological orderings of the provided directed acyclic graph. Returns an error if the given graph is undirected or cyclic. The function must be implemented non-recursively.
