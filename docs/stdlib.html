Package stdlib implements common data structures and algorithms.

Restrictions

stdlib source code can only use the following packages: 

errors
fmt
sort
strings
strconv
math
math/rand
math/bits
container/heap
container/list

Imports from hsecode.com/stdlib itself are allowed.

Check "Restrictions" section in subpackages documentation for other syntactic limitations.
